# update-delivery-suggester

定例MTG後のアポ打診・新機能デリバリーを支援するAIスキル。
メディカルフォース（電子カルテ）のアップデート情報とSalesforceのMTGメモを照合し、
確定／候補／対象外の3段階で整理する。最終的なデリバリー文面まで出力する。

---

## 使い方

### インプットの準備（2つ）

**① アップデート情報.md**
メディカルフォースの最新アップデート情報ファイルをチャット画面にアップロードする。

**② SalesforceのMTGメモ**
以下のフィールドが写っているスクショを撮る：
- 議事録サマリ（決定事項・ペンディング事項・ネクストアクション）
- 先方課題
- 持ち帰り事項

### Claude Projectへの入力

1. `update-delivery-suggester` プロジェクトを開く
2. アップデート情報.mdをアップロード
3. SFスクショを貼り付ける
4. 顧客名とSFのレコードURLを添えて送信する

例：
```
恵比寿ブライトスキンクリニック
SF：https://xxxxx

このデータからお願い
```

### 候補の確認ステップ

AIから「候補（まとめて確認）」が出たら、件ごとに以下を返信する：
- 使える → 「確定に格上げ」
- 案内済み → 「○番は案内済み」
- 不要 → 「○番は対象外」
